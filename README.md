Ба ҳама салом. Ман аз Тоҷикистон ҳастам. Ин як бозии инди мебошад, яъне таҳиягар танҳо худам ҳастам. Ман мехоҳам, ки бачаҳои имрӯза фаҳманд, ки бозиҳои пештара чӣ гуна буданд.

Бозиҳои пешина осон набуданд ва бачаҳоро тарбия мекарданд. Яъне, пештар ҳама бозиҳо душвор буд, мисли ҳаёт: то ҳаракат накунӣ, ҳеҷ ҷо намерасӣ.

Аммо имрӯз бисёр бозиҳо хеле осон шудаанд. Аз ҳамин сабаб, бачаҳо вақте ки бо мушкилӣ рӯ ба рӯ мешаванд, зуд таслим мешаванд.
